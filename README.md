# Gestion-del-desarrollo-de-sistemas-informaticos
Trabajo grupal realizado en la materia GDSI en la facultad de ingeniería (UBA). Cátedra Fontela 2C2025.

Integrantes: Franco Guardia (@franguardia12), Lucas Jang (@lucasjang01), Kevin Vallejo (@Kevin00404), Facundo Kresta (@facukresta), Nicolás Zysman (@NicolasZysman), Gianluca Negrotti (@GianlucaNegrotti)
